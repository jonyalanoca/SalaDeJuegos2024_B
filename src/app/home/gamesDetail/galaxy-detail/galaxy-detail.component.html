Guerra de las Galaxias

Caza de Naves es un emocionante juego de reflejos donde tendrás que eliminar la mayor cantidad de naves enemigas posible antes de que se acabe el tiempo. Usa tu ratón para aplastar las naves que aparecen en la pantalla, pero cuidado, ¡a medida que pase el tiempo las naves se moverán cada vez más rápido! ¿Tienes lo necesario para ser el mejor cazador de naves?

¡Acumula la mayor cantidad de puntos antes de que el tiempo se acabe!


Instrucciones:

  1. Tendrás 2 minutos para eliminar la mayor cantidad de naves enemigas posible.
  2. Elige la opción que creas correcta.
  3. Por cada nave que elimines, sumarás puntos.
  4. Las naves aparecerán y se moverán más rápido conforme avance el tiempo, aumentando el desafío.
